Days 0 - 2: Math, NumPy, and Pandas

Day 0: The Bridge to AI (Math Intuition)

1. Linear Algebra

  • Scalar: Single number (e.g., Age)
  • Vector: 1D list (e.g., [Age, Salary])
  • Matrix: 2D grid (The entire dataset)

The Dot Product (Predictions):

Combines inputs and weights.

$$y = (\vec{x} \cdot \vec{w}) + b$$

2. Calculus (Learning)

  • Loss: Model error (lower is better).
  • Gradient: Vector of derivatives pointing toward steepest error increase.
  • Learning Rate: Step size.

Gradient Descent:

Step in the opposite direction of the gradient.

$$W_{new} = W_{old} - (G \times LR)$$

3. Statistics

  • Mean: The central average.
  • Std Dev: Consistency/Spread.
  • Why Normalize? Levels the playing field so Gradient Descent is smooth.

Normalization (Z-Score):

$$x_{norm} = \frac{x - \mu}{\sigma}$$

Day 1: Tensors & Vectorization (NumPy)

1. Tensor Shapes & Broadcasting

Compare dimensions right-to-left. They must be equal, or one must be 1 (which then stretches).

import numpy as np

# A 3D tensor: (38 matches, 11 players, 3 stats)
team_stats = np.random.rand(38, 11, 3)

# A 1D tensor: (3 league averages)
league_avg = np.array([85.0, 2.0, 10.0])

# Broadcasting stretches the (3,) over the (38, 11, 3)
diff = team_stats - league_avg

2. Matrix Multiplication (Deep Learning)

Uses the @ operator. Rule: Inner dimensions must match. Output takes outer dimensions.

# Shape: (10000 matches, 15 features)
X = np.random.rand(10000, 15)

# Shape: (15 input features, 64 neurons)
W = np.random.rand(15, 64)

# Inner 15s match -> Output shape: (10000, 64)
predictions = X @ W

Day 2: Data Manipulation (Pandas)

1. Slicing & Boolean Filters

  • iloc: strictly Integer positions.
  • loc: strictly Label names.
# 1. Label Extraction
names_goals = df.loc[:, ["Player", "Goals"]]

# 2. Boolean Filtering (Note the parenthesis & bitwise |)
elite = df[(df["Goals"] > 20) | (df["Assists"] > 15)]

2. Handling Chaos (NaN)

Neural networks crash on NaN. Must drop or fill.

# Drop any row containing a NaN
df = df.dropna(axis=0)

# Fill missing distance with the column average
avg = df["Distance_km"].mean()
df["Distance_km"] = df["Distance_km"].fillna(avg)

3. GroupBy (Split-Apply-Combine)

Compress event data into strategic summaries.

# Group by Club, calculate mean of multiple columns
team_summary = df.groupby("Club").agg({
    "Distance_km": 'mean',
    "Pass_Accuracy": 'mean'
})

4. Feature Engineering

Creating new signals to make learning easier.

import numpy as np

# 1. Mathematical Ratio
df["Pass_Acc"] = df["Completed"] / df["Attempted"]

# 2. Conditional Binarization (1 or 0)
df["High_Work_Rate"] = np.where(df["Dist"] > 11.5, 1, 0)
