Day 5: Unsupervised Learning

K-Means, The Elbow Method, and PCA Compression

1. K-Means Clustering

Groups unlabelled data into K distinct clusters by minimizing the distance between points and their Centroids. There is NO y variable!

from sklearn.cluster import KMeans

# Initialize with desired number of clusters
kmeans = KMeans(n_clusters=3, random_state=42)

# Fit the model AND get the group labels instantly
labels = kmeans.fit_predict(X_scaled)

2. The Elbow Method (Finding K)

Tests multiple values of K to find the optimal balance point (the "elbow" of the graph) where diminishing returns begin.

# Loop through K=1 to K=10
inertia_scores = []
for k in range(1, 11):
    kmeans = KMeans(n_clusters=k)
    kmeans.fit(X_scaled)
    # Append the calculated Inertia score
    inertia_scores.append(kmeans.inertia_)

# Plot inertia_scores to find the 'Elbow'

3. PCA (Dimensionality Reduction)

Squashes high-dimensional data (e.g., 50 stats) down to a lower dimension (e.g., 2D) so humans can visualize the clusters.

from sklearn.decomposition import PCA

# Tell PCA we want 2 total components (PC1, PC2)
pca = PCA(n_components=2)

# Squash the data
X_2D = pca.fit_transform(X_scaled)

4. Mandatory Scaling

Golden Rule: You MUST scale your data before using PCA or K-Means. Because they rely on physical geometry and variance, large numbers (like Salaries) will completely destroy the math if left unscaled.

from sklearn.preprocessing import StandardScaler

scaler = StandardScaler()
X_scaled = scaler.fit_transform(X_raw)

# Now pass X_scaled into PCA and KMeans
