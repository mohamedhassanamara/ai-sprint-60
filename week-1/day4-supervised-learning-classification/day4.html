Day 4: Classification

LogReg, Confusion Matrices, Trees, and Forests

1. Logistic Regression & Probabilities

Uses the Sigmoid curve to squash outputs between 0.0 and 1.0. Perfect for binary targets (Win/Loss, Healthy/Injured).

from sklearn.linear_model import LogisticRegression

model = LogisticRegression()
model.fit(X_train, y_train)

# Hard prediction (0 or 1) based on a threshold (default 0.5)
hard_pred = model.predict(X_new)

# Raw probabilities [Probability of 0, Probability of 1]
probs = model.predict_proba(X_new)

2. The Confusion Matrix

Accuracy is a trap on imbalanced datasets. Use the matrix to track exact error types.

  • TP (True Positive): Correctly predicted the target.
  • TN (True Negative): Correctly predicted the baseline.
  • FP (False Positive): False Alarm (Type I Error).
  • FN (False Negative): The Miss (Type II Error - usually the most dangerous).
from sklearn.metrics import confusion_matrix
cm = confusion_matrix(y_true, y_pred)

3. Precision, Recall, & F1

  • Precision: "Quality" - When I call a positive, am I right? Reduces FP.
  • Recall: "Quantity" - Did I catch all the actual positives? Reduces FN.
  • F1-Score: The harmonic mean (balance) of Precision and Recall.
from sklearn.metrics import classification_report

# Generates a complete table of Precision, Recall, and F1
report = classification_report(y_true, y_pred)
print(report)

4. Trees & Random Forests

  • Decision Tree: Fast, logical, but highly prone to Overfitting. Limit depth to protect it.
  • Random Forest: An ensemble of trees. Each gets a random subset of data and features. Their majority vote cancels out individual errors.
from sklearn.ensemble import RandomForestClassifier

# n_estimators = number of trees in the forest
model = RandomForestClassifier(n_estimators=100, max_depth=5)
model.fit(X_train, y_train)
y_pred = model.predict(X_test)
